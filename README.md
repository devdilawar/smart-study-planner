# Smart Study Planner

A modern, responsive, and feature-rich study planner built using **HTML, CSS, and JavaScript**. This application helps students organize their academic life by managing tasks, tracking study sessions, monitoring goals, calculating GPA, and analyzing productivity through an elegant and intuitive interface.

---

## Features

### Dashboard Overview

* Personalized dashboard
* Daily productivity summary
* Task completion statistics
* Focus time tracking
* Goal progress overview
* Study streak counter
* Weekly activity chart
* Quick access to tasks, habits, and subjects

### Task Management

* Create, edit, and delete tasks
* Set task priorities
* Assign subjects to tasks
* Add descriptions
* Set due dates and due times
* Filter and sort tasks
* Mark tasks as completed

### Focus Timer (Pomodoro Technique)

* Focus sessions
* Short and long break modes
* Custom timer durations
* Subject-linked study sessions
* Session history
* Daily focus time tracking

### Calendar

* Interactive monthly calendar
* Navigate between months
* View tasks by selected date
* Today's quick navigation
* Schedule visualization

### Subject Management

* Create and manage subjects
* Assign custom colors
* Store instructor information
* Manage credit hours
* Connect subjects with tasks, notes, goals, and GPA

### Notes

* Create, edit, and delete notes
* Organize notes by subject
* Store lecture summaries
* Save important reminders

### Goals

* Create academic goals
* Track progress
* Set target values
* Set deadlines
* Visual progress indicators

### Habit Tracker

* Create daily habits
* Daily and weekday tracking
* Mark habits as completed
* Track study streaks
* Build consistent routines

### Analytics

* Focus time trends
* Subject-wise study distribution
* Task completion statistics
* Habit consistency analysis
* Productivity insights

### GPA Calculator

* Add unlimited courses
* Enter course credit hours
* Select letter grades
* Automatic GPA calculation
* Standard 4.0 GPA scale
* Credit hour summary

### Settings

* Update display name
* Toggle light and dark themes
* Export application data
* Import saved data
* Reset all locally stored data

## Data Storage

All application data is stored locally using the browser's Local Storage.

Stored data includes:

* Tasks
* Subjects
* Notes
* Goals
* Habits
* Focus Sessions
* GPA Data
* User Preferences

No external database or backend service is required.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/smart-study-planner.git
```

Open the project folder:

```bash
cd smart-study-planner
```

Run the project by opening the `index.html` file in your browser.
